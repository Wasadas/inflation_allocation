# Inflation Allocation

A research project scaffold for studying portfolio allocation under inflation regimes.

## Goals
- Define inflation regimes (e.g., high/low, rising/falling)
- Backtest allocation rules across regimes
- Compare with simple benchmarks

## Structure (planned)
- `notebooks/` exploration
- `src/` reusable code
- `data/` raw/intermediate data (not committed)
- `outputs/` figures/tables (not committed)
